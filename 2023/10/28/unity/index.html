AssetStudio 是一个用于探索、提取和导出资产和资产包的工具

功能

  • Texture2D : convert to png, tga, jpeg, bmp
  • Sprite : crop Texture2D to png, tga, jpeg, bmp
  • AudioClip : mp3, ogg, wav, m4a, fsb. support convert FSB file to WAV(PCM)
  • Font : ttf, otf
  • Mesh : obj
  • TextAsset
  • Shader
  • MovieTexture
  • VideoClip
  • MonoBehaviour : json
  • Animator : export to FBX file with bound AnimationClip

开始 .Unity3d 探索之路

.Unity3d

.Unity3d 文件是一个3D游戏档案，由安装了Unity web Player插件的Web浏览器使用，用于在线玩Unity开发的3D游戏。它包含资源，例如模型、纹理、动画和声音，以及其他在Unity中从eb浏览器使用，用于在线玩Unity开发的3D游戏。它包含资源，例如模型、纹理、动画和声音，以及其他在Unity中从.Unity 场景文件构建的游戏数据

操作

1. 首先，我们需要下载游戏安装包，切记该游戏必须是 Unity 开发的

注意：以后的都是这样操作，如果有更换步骤，本教程将会更新

2. 将安装包进行解压

3. 解压后的安装包有 data.unity3d 那么游戏是 Unity 开发

参照目录

  • 根文件夹 -> assets -> bin -> Data

注意：路径可能会有变化，以实际情况为主

4. 打开AssetStudio工具，把下载的apk文件拖拽到AssetsStudio

5. 导出素材，在AssetStudio中，有两种方式可以导出素材

  • Export -> All Assets，导出全部，工具会自动将内容放入对应的文件夹
